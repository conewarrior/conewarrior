# 사용자 경험을 설계합니다. AI로 효율을, 데이터로 근거를,시스템으로 일관성을.

<br />
<br />

AI로 개발하는 프로덕트 디자이너입니다.

Claude Code를 주 도구로 사용하며, 데이터 분석부터 디자인 시스템 설계, 프로토타입 제작,Figma 플러그인, MCP 서버, npm 패키지 개발까지 전방위적으로 다루고 있습니다.

이 GitHub은 프로덕트 디자이너가 더 빠르고 정확하게 일할 수 있는 방법을 찾는 실험장입니다.

<br />
